# Guilherme Porto — Portfólio

Site pessoal de página única, bilíngue (PT/EN), com timeline de experiência, projetos e contato.

**🔗 Ver ao vivo:** https://gui-porto.github.io/

## Sobre

Portfólio construído para apresentar minha trajetória como Supervisor Administrativo em transição para desenvolvimento web, incluindo experiência profissional, formação e o projeto **Lexo** (SaaS jurídico).

## Funcionalidades

- 🌐 **PT/EN** — troca de idioma com um clique, sem recarregar a página
- ✍️ **Efeito de digitação** no hero, com frases rotativas
- 🎬 **Animações de entrada** ao rolar a página (`IntersectionObserver`)
- 📱 **Responsivo** — menu mobile e layout adaptado para qualquer tela
- 📄 Link direto para currículo em PDF e contato (e-mail, telefone, redes sociais)

## Tecnologias

HTML5 · CSS3 · JavaScript (vanilla, sem frameworks nem build step)

## Estrutura

\`\`\`
index.html      # markup e conteúdo (PT/EN)
css/style.css   # estilos e animações
js/main.js      # troca de idioma, digitação, reveal ao rolar
assets/         # foto de perfil e currículo em PDF
\`\`\`

## Rodando localmente

Não há dependências nem build — basta abrir o `index.html` no navegador, ou servir a pasta:

\`\`\`bash
npx serve .
\`\`\`
